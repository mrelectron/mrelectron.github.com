MT4 Keyboard Shortcuts

May 8th, 2012

This post lists MT4 keyboard shortcuts, as sometimes it is a pain to swap between mouse and keyboard.

Scroll

  • Left : Left
  • Left Fast : Page Up
  • Left OneBar : F12
  • Right : Right
  • Right Fast : Page Down
  • Right OneBar : Shift+F12
  • End : End
  • Start : Home

Windows

  • Data : Ctrl+D
  • Indicators : Ctrl+I
  • Market Watch : Ctrl+M
  • Navigator : Ctrl+N
  • Terminal : Ctrl+T
  • Tester : Ctrl+R

Switch

  • Chart Next : Ctrl+F6
  • Profile Next : Ctrl+F5

Dialogs

  • Chart Props : F8
  • Global Vars : F3
  • History Center: F2
  • New Order : F9
  • Open Windows : Alt+W
  • Options : Ctrl+O
  • Prices Dialog : F10
  • Print : Ctrl+P

Toggle

  • Bar Chart : Alt+1
  • Candle Chart : Alt+2
  • Fast Nav : Enter
  • Full screen : F11
  • Grid : Ctrl+G
  • Line Chart : Alt+3
  • OHLC : Ctrl+H
  • Period Seps : Ctrl+Y
  • Volume : Ctrl+L

Miscellany

  • Close Dialogs : Esc
  • Close Window : Ctrl+F4
  • Copy : Ctrl+C
  • Crosshair : Ctrl+F
  • Help : F1
  • MetaEditor : F4
  • Quit : Alt+F4
  • Save : Ctrl+S
  • Undo : Ctrl+Z
  • Zoom In : +
  • Zoom out : -

There is no facility within MT4 to assign keyboard shortcuts. But if you want to expand the range of keyboard shortcuts then you can use a free utility called AutoHotKey, you can then go crazy with keyboard shortcuts.
